# 核心文档索引 (Core Documentation Index)

## 目录说明

本目录存放 Echo Server 的核心技术文档，所有 AI Agent 和开发者必须严格遵守文档管理规范。

### 📁 目录结构

- **changes/**: 代码变更记录（最重要）
  - `CHANGELOG.md`: 变更总览
  - `features/`: 新增功能记录
  - `bugfixes/`: Bug 修复记录
  - `optimizations/`: 性能优化记录
  - `merge-reports/`: 上游合并报告

### 🚫 保护规则 (Hard Rules)

1. **禁止删除**: 严禁删除本目录下的任何文档。
2. **禁止移动**: 严禁将文档移动到其他目录。
3. **必须记录**: 任何代码变更必须对应一条变更记录。
4. **必须索引**: 新增文档必须在 CHANGELOG.md 中索引。
