## 🎯 使用指南

### 对于 AI Agent

1. **首次接触项目时**：
   - 先读 `ECHO执行方案-精简版.md` 了解项目宪法
   - 再读 `AGENTS.md` 了解具体执行规范

2. **日常开发时**：
   - 遇到技术决策问题 → 查阅 `ECHO执行方案-精简版.md`
   - 遇到规范问题 → 查阅 `AGENTS.md`

3. **判断方案是否可行时**：
   - 使用项目宪法中的"最终判断标准"
   - 以"10 年后还能维护"为目标

### 对于开发者

1. **开始任何工作前**：

   - 理解核心约束和禁止事项

2. **技术方案设计时**：
   - 遵循"正确性 > 完整性 > 性能 > 开发速度"原则
   - 确保状态一致性和可回放性

3. **代码提交前**：
   - 检查是否违反了项目宪法
   - 确保符合 AGENTS.md 中的规范
